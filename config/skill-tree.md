# Antigravity Skill Tree

A compact index of the active Antigravity specialist library.

## Engineering

- `agency-software-architect`: top-level system shape, boundaries, tradeoffs
- `agency-backend-architect`: backend services, APIs, request flows
- `agency-frontend-developer`: app UI implementation
- `agency-devops-automator`: CI/CD and deployment automation
- `agency-database-optimizer`: schema, indexes, query performance
- `agency-code-reviewer`: workflow-first diff and PR review
- `agency-api-tester`: API contracts, integration edges, error handling, performance
- `agency-mcp-builder`: MCP tool/resources/prompts and server contracts
- `agency-mobile-app-builder`: native and cross-platform mobile work
- `agency-ai-engineer`: production AI/ML implementation

## Delivery and planning

- `agency-project-shepherd`: executable delivery plans, dependencies, risks
- `agency-sprint-prioritizer`: sprint and backlog ordering
- `agency-studio-producer`: portfolio strategy and investment tradeoffs
- `agency-product-manager`: roadmap, discovery, lifecycle decisions
- `agency-experiment-tracker`: experiments and hypothesis tracking

## Security and reliability

- `agency-security-engineer`: threat review and secure implementation
- `agency-sre-site-reliability-engineer`: SLOs, observability, reliability posture
- `agency-incident-response-commander`: incident handling and postmortems
- `agency-compliance-auditor`: audit and control readiness
- `agency-legal-compliance-checker`: regulatory and policy review

## Data and documents

- `agency-data-engineer`: reproducible data pipelines and analytics-ready assets
- `agency-document-generator`: PDF, PPTX, DOCX, XLSX generation
- `agency-analytics-reporter`: stakeholder-facing analytics and KPI reporting
- `agency-executive-summary-generator`: decision-oriented executive summaries

## Design and quality

- `agency-ui-designer`: visual systems and interface direction
- `agency-ux-architect`: UX structure and implementation patterns
- `agency-ux-researcher`: user research and usability insight
- `agency-accessibility-auditor`: WCAG and inclusive-design review
- `agency-evidence-collector`: evidence-heavy QA and visual verification
